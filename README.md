## ⭐ Projetos em Destaque

### 🤖 [RAG - Bot de Resolução de Erros](https://github.com/Gabriela-S2/RAG)
> Chatbot inteligente baseado em **RAG (Retrieval-Augmented Generation)** desenvolvido em PHP.  
> Lê ficheiros `.txt`, indexa-os com **pgvector** e responde perguntas com contexto real  
> via **OpenRouter** — com streaming de respostas em tempo real via **SSE**.  
> `PHP` `PostgreSQL` `pgvector` `OpenRouter` `LLPhant` `SSE`

---

### 💬 [Chatbot Inteligente WhatsApp](https://github.com/Gabriela-S2/chatbot-inteligente-whatsapp)
> Plataforma de atendimento com **IA para triagem automática** + interface web  
> para atendimento humano com múltiplos setores, histórico e envio de ficheiros.  
> `Python` `IA` `WhatsApp`

---

### 🚌 [Cadê o Ônibus?](https://github.com/Gabriela-S2/cade_o_onibus)

> `Python` `Flask` `Leaflet.js` `SQLite` `Pandas`

---
